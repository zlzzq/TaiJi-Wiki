# 39深度思考

## Summary

若（道）心有住，即为非住

## Key Claims

- 若（道）心有住，即为非住
- 若（俗）心无住，即为是住
- 所以三晳可以格解一切经典

## Teaching Use

本文适合用于 `待归类` 的学习路径中，重点回答：什么是“心”？；什么是“性”？；什么是“三晳”？

## Common Misreadings

- 但中医困境太难搞了，比如药材，现在很多都是种植的，药性就差很多
- 这可能是常规思维的惯性，我有时候参悟或读《太极经》，忽然生起某种觉受，但一注意，头脑一推敲，那种感觉就渐渐消失了
- （先斩断修行的外围障碍，以便能静下心来）眼睛看见五彩缤纷的世界万象时，眼睛没有乱；耳朵听见各种各样的声音时，耳朵也没乱；但是，心在处理这些信息时却乱了，这是为什么？

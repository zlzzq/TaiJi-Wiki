# 26相信自己

## Summary

读《26相信自己》时，先抓住它围绕“悟”给出的入口，再分辨它是在立结构、破误读，还是把理路导向练习。 本页 v2 编译重点是把材料转成可复述、可反问、可练习的学习入口，而不是替代原文。

## Core Problem

本文在“待归类”中处理“悟”如何被理解和使用的问题：它不是要求先背定义，而是把学习者带回具体问题、材料语境和自身反问。

# Root Doctrine

## Summary

根本图不是知识树，而是三晳思维的走法：从有极界可说之理入手，经太极界有无圆转，知其不可用旧知占有无极界。

```mermaid
flowchart LR
  A["有极界 / 有界"] --> B["太极界 / 有无界"]
  B --> C["无极界 / 无界"]
  A --> D["生成"]
  D --> E["对待"]
  E --> F["变化"]
  F --> D
  B --> D
  C --> B
```

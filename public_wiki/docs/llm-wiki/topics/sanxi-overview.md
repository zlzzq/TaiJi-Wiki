# 三晳总览

## Summary

三晳是把万有放回生成、对待、变化三路中观察，并借此转变思维的核心入口。 v2 页面把它整理成问题、证据、概念关系和练习四层，避免停留在摘句集合。

## Core Problem

本专题要解决的是：学习者如何在“三晳总览”上形成稳定说法，同时保留材料中的不同口径，不把某一句话立成唯一结论。

## Why This Matters

它是整套资料的公共入口；不先把三晳讲活，后面的三界、本知、理入都容易变成名词堆砌。

## Misreadings

- 把一个教学口径说成唯一绝对口径。
- 只摘句不看结构，导致材料之间互相打架。
- 把概念页当结论页，不再回到练习和反问。

## Self-QA Lesson

自问：这个专题先解决什么问题？

自答：先抓住“三晳是把万有放回生成、对待、变化三路中观察，并借此转变思维的核心入口。”，不要先把它讲成术语表。

自问：我该怎样检查自己有没有讲死？

自答：看自己是不是只选了一个口径，然后把其他材料都裁掉。

自问：这一页怎么练？

自答：先用一句白话复述，再回到 Source Matrix 找一条证据，最后反问自己的说法哪里可能过头。

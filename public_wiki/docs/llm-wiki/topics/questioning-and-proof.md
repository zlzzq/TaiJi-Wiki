# 问道与证道

## Summary

问道不是索取答案，证道也不是自我声明，而是经得起反问和印证。 v2 页面把它整理成问题、证据、概念关系和练习四层，避免停留在摘句集合。

## Core Problem

本专题要解决的是：学习者如何在“问道与证道”上形成稳定说法，同时保留材料中的不同口径，不把某一句话立成唯一结论。

## Why This Matters

它处理学习者最常见的求答案、求确认和自封境界问题。

## Misreadings

- 把一个教学口径说成唯一绝对口径。
- 只摘句不看结构，导致材料之间互相打架。
- 把概念页当结论页，不再回到练习和反问。

## Self-QA Lesson

自问：这个专题先解决什么问题？

自答：先抓住“问道不是索取答案，证道也不是自我声明，而是经得起反问和印证。”，不要先把它讲成术语表。

自问：我该怎样检查自己有没有讲死？

自答：看自己是不是只选了一个口径，然后把其他材料都裁掉。

自问：这一页怎么练？

自答：先用一句白话复述，再回到 Source Matrix 找一条证据，最后反问自己的说法哪里可能过头。

## Practice

- 用 80 字以内说明“问道与证道”解决什么问题。
- 从 Source Matrix 任选两篇资料，说明它们分别提供了哪一层证据。
- 写一句容易说死的话，再把它改成更稳的说法。

# 变化 / 流行 的口径差异

## Summary

本页不裁定某个说法唯一正确，而是记录不同口径在何处使用，防止学习者拿一种讲法压掉另一种讲法。

## The Tension

变化、流行、流易 指向同一问题域，但可能分属传承旧名、现代转写、理路说明或练习入口。

## Source Positions

- 旧口径多保留传承味。
- 新口径多服务现代理解。
- 练习口径更重视能否让学习者参得进去。

## Stable Reading

稳读法是先问这句话在解决什么问题，再问它立在哪一界，最后问它怎样转入生、对、变的圆转。

## When To Use Which Term

- 讲传承出处时保留原术语。
- 讲入门练习时使用更易懂的说法。
- 讲口径差异时把两者并列，不强行抹平。

## Practice Questions

- 任选两个说法，分别说明它们服务哪一类问题。
- 把唯一正确改写为某一语境下更稳。

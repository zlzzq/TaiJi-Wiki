# 心

## Summary

“心”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：心在这套资料里不是单一心理学意义上的“心”，而是与本心、当来、知、应、住执等问题连在一起。 - 功能定位：心既可能成为障碍，也可能成为转机；执于心，则成见碍，返于本心，则近道。 - 易混点：不能把“心”直接等同于“性”；也不能把“心”只当情绪和念头。 - 主要出处：`09问道证道.docx`、`11一体两应.docx`、`16心本无象.docx`、`35心外无物.docx

## Usage Scenarios

- 当问题涉及“心”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“心”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“心”，不引用原句。
- 举一个例子说明“心”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

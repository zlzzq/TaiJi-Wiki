# 证

## Summary

“证”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：证偏向印证、实证、住持与经得起反问，不是嘴上说“我悟了”就算。 - 课堂口径：理入偏“悟道”，行入偏“实证”；但真正的证，又不能离开悟。 - 易混点：资料很强调“印证的是相似，不是一模一样”；所以不能拿别人的语言硬套自己的境界。 - 主要出处：`附1理入行入.docx`、`50知者不问.docx`、`09问道证道.docx

## Usage Scenarios

- 当问题涉及“证”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“证”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“证”，不引用原句。
- 举一个例子说明“证”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

# 三界

## Summary

主口径：三界是太极学中安放生命、认知与境界层次的基本框架，常用教学口径为“无界、有无界、有界”

## Definition By Layer

- 主口径：三界是太极学中安放生命、认知与境界层次的基本框架，常用教学口径为“无界、有无界、有界”。
- 一知三界口径：在 `02一知三界.pdf` 与 `08客知本知.docx` 的讲法里，又常把“知”放在中枢来看：知下偏客知，知即本知，知上连性灵。
- 易混点：三界不是单纯空间分区，而是认知、生命、境界三者交织的分层结构。
- 主要出处：`02一知三界.pdf`、`09问道证道.docx`、`18道通三界.docx`、`33三晳讲论.pdf`。

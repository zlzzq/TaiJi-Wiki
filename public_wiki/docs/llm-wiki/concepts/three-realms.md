# 三界

## Summary

“三界”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：三界是太极学中安放生命、认知与境界层次的基本框架，常用教学口径为“无界、有无界、有界”。 - 一知三界口径：在 `02一知三界.pdf` 与 `08客知本知.docx` 的讲法里，又常把“知”放在中枢来看：知下偏客知，知即本知，知上连性灵。 - 易混点：三界不是单纯空间分区，而是认知、生命、境界三者交织的分层结构。 - 主要出处：`02一知三界.pdf`、`09问道证道.docx`、`18道通三界.docx`、`33三晳讲论.pdf

## Usage Scenarios

- 当问题涉及“三界”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

# 灵

## Summary

“灵”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：灵偏向应、动、用、发用的一侧，常与性配成“性体、灵用”来看。 - 教学口径：若只讲“灵很玄”，那是空话；要把它放回“性静无、灵阳有、一体两用”的结构里。 - 易混点：性与灵的安放位置，在哲理讲法与实证讲法里会有轻微差别，不能偷换层次。 - 主要出处：`09问道证道.docx`、`02一知三界.pdf`、`30三灵无上.docx

## Usage Scenarios

- 当问题涉及“灵”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“灵”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“灵”，不引用原句。
- 举一个例子说明“灵”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

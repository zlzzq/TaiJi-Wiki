# 三晳

## Summary

主口径：三晳不是只让人背三个名词，而是借三套互转之理去格解万有、破除见执、建立太极思维的方法

## Definition By Layer

- 主口径：三晳不是只让人背三个名词，而是借三套互转之理去格解万有、破除见执、建立太极思维的方法。
- 公开口径：显三晳常讲为“化生、对待、流行”；在后期整理材料中，也常转成“生成、对待、变化”，进一步简化为“生、对、变”。
- 易混点：不能把三晳只当常识定义；更不能把“听来的三晳”当成“自己已经透破的三晳”。
- 主要出处：`01三晳九问.docx`、`41台版谈道.docx`、`43基础知识.docx`、`50知者不问.docx`、`12前后三晳.docx`。

# 理入

## Summary

主口径：理入是通过理、见、思辨、参悟来入道的一路，在太极学公开阶段常以三界九晳、尤其显三晳为基础

## Definition By Layer

- 主口径：理入是通过理、见、思辨、参悟来入道的一路，在太极学公开阶段常以三界九晳、尤其显三晳为基础。
- 功能定位：附录材料直说“理入修心神”，重点是破见执、开智慧、走悟道之路。
- 易混点：理入不是背知识点；若只积累概念而不转成自己的思维，仍然走不进去。
- 主要出处：`01三晳九问.docx`、`附1理入行入.docx`、`55讲义第二.docx`、`50知者不问.docx`。

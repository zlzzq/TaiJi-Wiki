# 对待 / 相待

## Summary

“对待 / 相待”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：第二晳早期多写“相待”，后为便于理解，教学上常讲“对待”。 - 核心含义：不是只有“两端相对”，还包含“对外有待”的关系性；所以它不只是对立，也是互立、互待、互转。 - 易混点：只懂“相对”，不懂“相待”，就容易把对待讲死，落在两边打架。 - 主要出处：`43基础知识.docx`、`01三晳九问.docx`、`11一体两应.docx

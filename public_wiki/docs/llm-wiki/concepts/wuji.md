# 无极

## Summary

主口径：无极常与无界、元真、本源相连，也出现在功行名相如“十方无极裆”“一本无极桩”中

## Definition By Layer

- 主口径：无极常与无界、元真、本源相连，也出现在功行名相如“十方无极裆”“一本无极桩”中。
- 分层理解：若从三界理说看，无极更接近无界之根；若从功行讲，则又成为实际入手门径的名称。
- 易混点：同一个“无极”，既可讲哲理层次，也可讲功行起点，不能混作一谈。
- 主要出处：`02一知三界.pdf`、`09问道证道.docx`、`50知者不问.docx`。

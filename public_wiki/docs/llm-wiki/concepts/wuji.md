# 无极

## Summary

“无极”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：无极常与无界、元真、本源相连，也出现在功行名相如“十方无极裆”“一本无极桩”中。 - 分层理解：若从三界理说看，无极更接近无界之根；若从功行讲，则又成为实际入手门径的名称。 - 易混点：同一个“无极”，既可讲哲理层次，也可讲功行起点，不能混作一谈。 - 主要出处：`02一知三界.pdf`、`09问道证道.docx`、`50知者不问.docx

## Usage Scenarios

- 当问题涉及“无极”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“无极”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“无极”，不引用原句。
- 举一个例子说明“无极”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

# 前后三晳

## Summary

“前后三晳”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：前三晳是前人传下来的“化生、对待、流行”这一路传承成果；后三晳是自己参、自己化、自己通以后真正立起来的三晳。 - 核心判断：资料说得很直，后三晳必须是你自己的；它像桥，没这个桥，你和三晳还是两端。 - 教学意义：老师讲给你的，最多帮你入见；真正开见、悟见、出见，靠你立自己的后三晳来破前三晳。 - 主要出处：`12前后三晳.docx`、`附2前后三晳.docx`、`50知者不问.docx

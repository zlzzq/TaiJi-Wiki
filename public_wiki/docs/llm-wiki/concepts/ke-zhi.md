# 客知

## Summary

主口径：客知是后天学而知之、依六根与知解而成立的知

## Definition By Layer

- 主口径：客知是后天学而知之、依六根与知解而成立的知。
- 功能定位：客知“能知不能悟”，擅长明白万有、区别对象、处理世知与生存之知。
- 易混点：客知的明白仍落对待，所以“有所明白”同时也意味着“有所不明白”。
- 主要出处：`08客知本知.docx`、`09问道证道.docx`、`43基础知识.docx`。

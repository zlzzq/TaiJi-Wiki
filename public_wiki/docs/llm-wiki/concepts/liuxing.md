# 流行 / 变化

## Summary

主口径：第三晳所讲的是万有流转、往复、变易之理；在后期通俗教学中，常直接讲“变化”

## Definition By Layer

- 主口径：第三晳所讲的是万有流转、往复、变易之理；在后期通俗教学中，常直接讲“变化”。
- 课堂口径：流行不是一句“周行不殆”就讲完了，它强调的是始终相续、终而复始、变中有化。
- 易混点：把流行只当“变化很多”是不够的；它更重要的是说明变化如何成圈、如何回转。
- 主要出处：`01三晳九问.docx`、`20周行不殆.docx`、`43基础知识.docx`。

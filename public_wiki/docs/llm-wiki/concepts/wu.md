# 悟

## Summary

主口径：悟不是普通理解，而是进入觉悟境界后的“心中明白”

## Definition By Layer

- 主口径：悟不是普通理解，而是进入觉悟境界后的“心中明白”。
- 资料口径：`08客知本知.docx` 直说，常境中的感受与知解都不等于悟；悟要从迷中出、从有为中损、从境界中显。
- 易混点：把一时感受、语言漂亮、概念清楚，当成悟，都是资料反复提醒的错路。
- 主要出处：`08客知本知.docx`、`09问道证道.docx`、`50知者不问.docx`。

# 悟

## Summary

“悟”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：悟不是普通理解，而是进入觉悟境界后的“心中明白”。 - 资料口径：`08客知本知.docx` 直说，常境中的感受与知解都不等于悟；悟要从迷中出、从有为中损、从境界中显。 - 易混点：把一时感受、语言漂亮、概念清楚，当成悟，都是资料反复提醒的错路。 - 主要出处：`08客知本知.docx`、`09问道证道.docx`、`50知者不问.docx

## Usage Scenarios

- 当问题涉及“悟”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“悟”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“悟”，不引用原句。
- 举一个例子说明“悟”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

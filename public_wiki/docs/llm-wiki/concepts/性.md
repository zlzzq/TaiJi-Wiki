# 性

## Summary

“性”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：性常被置于更本、更静、更根的一侧，是生命本源与智慧根性的指向。 - 与灵关系：资料中常讲“性之与灵，一体两用”，即性不是孤零零一个抽象词，而要放在与灵的对应中看。 - 易混点：很多地方口头上说“见性”，但资料反复提醒，照抄古人名句并不能证明自己真见到了。 - 主要出处：`09问道证道.docx`、`16心本无象.docx`、`50知者不问.docx

## Usage Scenarios

- 当问题涉及“性”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

## Common Confusions

- 把“性”当成单个定义背下来。
- 把某个材料中的讲法当成所有场景的唯一标准。

## Practice Questions

- 用自己的话解释“性”，不引用原句。
- 举一个例子说明“性”容易被怎样误解。

## Teaching Pattern

先用白话定主轴，再说明材料口径，最后用反问检查是否把话说死。

# 性知

## Summary

“性知”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：在 `08客知本知.docx` 的“一知三界”讲法里，性知位于本知之上，是更根本的一层“知上之性灵”。 - 版本差异：`09问道证道.docx` 又特别提醒，在某些实证语境里，“性”和“灵”未必直接安在无界，说明哲理分层与实证分层不能混讲。 - 教学建议：讲概念框架时，可先说“有界客知、有无界本知、无界性知”；讲实证辨析时，必须补一句“这只是其中一路讲法”。 - 主要出处：`08客知本知.docx`、`09问道证道.docx`、`02一知三界.pdf

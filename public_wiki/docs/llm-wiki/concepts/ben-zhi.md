# 本知

## Summary

主口径：本知常被讲成“本如当来之知”“本心之知”，位于客知之上、又与觉悟更近

## Definition By Layer

- 主口径：本知常被讲成“本如当来之知”“本心之知”，位于客知之上、又与觉悟更近。
- 功能定位：资料中反复强调“本知悟而不能知”，即本知偏向觉与悟，不偏向对象化知解。
- 易混点：本知不是普通聪明，也不是概念上说一句“我懂本心了”；它要跟客知区分层次。
- 主要出处：`08客知本知.docx`、`09问道证道.docx`、`36三晳讲义.docx`。

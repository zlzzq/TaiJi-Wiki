# 太极

## Summary

主口径：太极既指整套太极理法的大框架，也可在特定语境里指“一知”“太极思维”“圆融运行”的层次

## Definition By Layer

- 主口径：太极既指整套太极理法的大框架，也可在特定语境里指“一知”“太极思维”“圆融运行”的层次。
- 与三晳关系：三晳是显三晳，是有界公开可讲的基础理路；把三晳走活，才有可能近太极思维的圆应。
- 易混点：不能把“太极”只缩成图案、术语或某一套动作；资料里它更是生命、思维、理法的总枢纽。
- 主要出处：`01三晳九问.docx`、`02一知三界.pdf`、`41台版谈道.docx`、`50知者不问.docx`。

# 太极

## Summary

“太极”在 v2 Wiki 中不是孤立定义，而是一个需要按资料层次、使用场景和误读风险来理解的学习节点。

## Definition By Layer

- 主口径：太极既指整套太极理法的大框架，也可在特定语境里指“一知”“太极思维”“圆融运行”的层次。 - 与三晳关系：三晳是显三晳，是有界公开可讲的基础理路；把三晳走活，才有可能近太极思维的圆应。 - 易混点：不能把“太极”只缩成图案、术语或某一套动作；资料里它更是生命、思维、理法的总枢纽。 - 主要出处：`01三晳九问.docx`、`02一知三界.pdf`、`41台版谈道.docx`、`50知者不问.docx

## Usage Scenarios

- 当问题涉及“太极”的定义时，先回到来源资料。
- 当不同资料口径不一致时，先区分教学阶段和使用场景。
- 当你准备对外说明时，先用白话讲，再补术语。

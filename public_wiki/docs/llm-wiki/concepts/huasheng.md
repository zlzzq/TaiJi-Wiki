# 化生 / 生成

## Summary

主口径：早期公开口径多说“化生”，后期为了把六字圈讲得更清楚，常改说“生成”

## Definition By Layer

- 主口径：早期公开口径多说“化生”，后期为了把六字圈讲得更清楚，常改说“生成”。
- 变化原因：`43基础知识.docx` 明讲，原来“化生”的“化”后来更多被归到“变化/流行”里，为了避免“化”和“变”缠在一起，就把“化生”调整成“生成/成生”的教学口径。
- 教学建议：若讲传承与原口径，用“化生”；若讲六字循环与现代讲义的清晰表达，可讲“生成”。
- 主要出处：`01三晳九问.docx`、`43基础知识.docx`、`41台版谈道.docx`。

# 对

## Summary

对 不是孤立定义，而是三晳资料中用于安放生起、对待、变化和三界层次的节点。读它时先看资料中怎样使用，再看自己是否把它按字面锁死。

## Definition By Layer

- 有极界：先以文字和例子识别 对。
- 太极界：看 对 与相关概念怎样互相转化。
- 无极界：承认 对 不能被一个定义完全占有。

## 三晳安放

- 生成：追问 对 从哪一类问题生起。
- 对待：辨明 对 破的是哪一种二边。
- 变化：看 对 怎样转入练习与修证。

## 使用场景

- 解释 对 的基础义。
- 辨别 对 与相邻术语的口径差异。
- 把 对 落到自问自答练习。
